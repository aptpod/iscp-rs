# iSCP-rs

iSCPv2 Client Library

## Installation

- Add dependency to Cargo.toml

    ```
    iscp-rs = "1"
    ```

## Usage

- [API Document](https://docs.rs/iscp-rs/latest/iscp/)
- [Example](./examples)

### Crate features

* `gen`: To regenerate protobuf messages for using latest protobuf version.
